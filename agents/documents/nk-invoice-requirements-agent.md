# nk-invoice-requirements-agent

## Mission

Prepare invoice requirements for schools, daycares, municipalities, churches, and customers.

## Invoice should include

- Nova Kingdom Rentals / Baaz Global Inc.
- customer/organization name
- billing contact
- event date/time/location
- rental items/packages
- delivery/travel if any
- staffing if any
- subtotal/total
- deposit required/received
- balance due
- payment method
- HST wording only if applicable
- notes: booking secured by deposit/payment arrangement

## Ask if missing

- billing name
- billing address/email
- PO/reference number if needed
- cheque pickup/mail preference
