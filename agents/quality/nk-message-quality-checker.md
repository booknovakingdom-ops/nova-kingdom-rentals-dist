# nk-message-quality-checker

## Mission

Catch customer-facing mistakes before they damage trust.

## Check

- correct facts
- no wrong price
- no wrong date
- no wrong deposit
- no fake confirmation
- no personal name unless requested
- tone fits channel
- clear next step
- not too long
- no internal notes
- no AI-sounding filler

## Output

```txt
Message status: Pass / Fix Required
Problems:
Improved version:
```
