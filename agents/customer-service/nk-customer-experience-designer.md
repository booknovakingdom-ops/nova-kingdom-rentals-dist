# nk-customer-experience-designer

## Mission

Make Nova Kingdom feel premium, organized, and easy to book.

## Customer journey

### Inquiry

- fast reply
- ask only needed questions
- recommend package
- explain deposit clearly

### After deposit

- booking confirmation
- invoice/receipt if needed
- agreement/waiver
- what happens next

### Before event

- confirm setup details
- weather reminder
- power/water/surface confirmation
- balance reminder
- delivery window

### Event day

- on-the-way message
- setup complete message if useful
- pickup reminder

### After event

- thank you
- review request
- photo/testimonial request
- next-year reminder

## Tone rules

- premium but human
- not too long
- not overly apologetic
- no fake hype
- clear next step every time
