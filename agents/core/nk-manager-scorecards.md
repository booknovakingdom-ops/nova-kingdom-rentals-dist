# Manager Scorecards

## Sales & Booking Manager
- New leads received
- Quotes sent
- Quote-to-deposit conversion
- Deposits received
- Stale leads needing follow-up
- Average booking value
- Discount leakage

## Customer Experience Manager
- Average response speed
- Messages corrected before sending
- Review requests sent
- Reviews received
- Complaints opened/resolved
- Customer proof/testimonials collected

## Operations Dispatch Manager
- Upcoming bookings
- Load checklists completed
- Staffing gaps
- Equipment maintenance flags
- Route conflicts
- Pickup/cleaning issues

## Safety Compliance Manager
- Events with safety review
- Yellow/red risk bookings
- Weather escalations
- Incident reports
- Damage claims
- Public/school event compliance checks

## Marketing Growth Manager
- Posts published
- Ads active
- Cost per qualified lead
- Cost per deposit
- Review/social proof assets created
- GBP updates
- Campaigns generating booked revenue

## Website SEO Manager
- Website conversion issues found
- Service-area pages created/updated
- Broken links/forms/buttons
- Price mismatch incidents
- Mobile UX issues
- SEO/content opportunities

## Finance Revenue Manager
- Revenue booked
- Cash collected
- Balances due
- Ad spend
- Gross margin estimate
- Equipment ROI progress
- Low-margin bookings flagged

## Documents Legal Manager
- Invoices sent
- Agreements sent/signed
- Waivers missing
- Insurance documents requested/sent
- Document expiries
- Legal/policy exceptions

## Quality Control Manager
- Outputs reviewed
- Outputs rejected
- Common mistake patterns
- Test cases failed
- Feedback-loop rules added

## Tech Automation Manager
- CRM updates completed
- Gmail drafts created
- Calendar entries created
- Website/GitHub tasks prepared
- Automation failures
- Owner approvals required
