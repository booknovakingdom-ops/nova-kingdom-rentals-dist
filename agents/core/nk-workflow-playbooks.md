# Nova Kingdom Rentals — Workflow Playbooks

## 1. New lead workflow

Trigger: any new inquiry from Facebook, Instagram, email, website, phone, school, or referral.

Steps:

1. Capture customer name/contact if available.
2. Extract known details: event date, address/town, time, event type, item/package, guest count, surface, power/water, attendant need.
3. Identify missing info.
4. Check availability if date and item are known.
5. Recommend product/package if customer is unsure.
6. Build quote only when enough info exists.
7. Run quote quality gate.
8. Draft reply.
9. CRM status: New Inquiry / Missing Info / Quote Sent.
10. Create follow-up task.

Minimum information before final quote:

- event date
- event address or town
- rental time/event time
- unit/package or event type
- surface type for setup
- water/power if needed

## 2. Quote workflow

1. Use official price from source of truth.
2. Calculate delivery: first 15 km free, then $0.72/km.
3. Add staff: $30/hour/person if needed.
4. Add package/lawn game upgrade.
5. Add processing fee if credit card/bank payment requested.
6. Calculate total.
7. Calculate 30% deposit.
8. Calculate balance.
9. Add next step: deposit secures booking.
10. Run quote quality gate.

Quote formula:

`Rental/package + travel + staffing + add-ons + approved fees = total`

`Deposit = total x 30%`

`Balance = total - deposit`

## 3. Deposit follow-up workflow

After quote sent:

- 4 hours: light check-in if customer seemed hot.
- 24 hours: remind date is not secured without deposit.
- 48 hours: offer to send invoice/agreement if they want to move ahead.
- 5 days: ask whether to keep quote open or close it for now.
- 14 days: cold lead revival.

Do not pressure schools/organizations the same way as casual Facebook leads; keep tone professional.

## 4. Confirmed booking workflow

Trigger: deposit received or institution-approved payment process accepted.

Steps:

1. Confirm payment/deposit.
2. Update CRM status to Confirmed Booking.
3. Create/send invoice or receipt if needed.
4. Send agreement/waiver.
5. Add booking to calendar.
6. Add dispatch task.
7. Add weather check task.
8. Add final balance reminder.
9. Add review request task after event.

## 5. Pre-event workflow

7 days before:

- confirm date/time/address/item
- confirm surface
- confirm power/water
- confirm remaining balance
- remind agreement/waiver if missing

3 days before:

- weather check
- delivery window
- access notes
- staff plan

1 day before:

- delivery/pickup reminder
- water shutoff reminder for wet units
- pickup access reminder
- final safety note

## 6. Dispatch workflow

For each confirmed event:

1. Verify address and route.
2. Calculate travel distance/time.
3. Confirm setup and pickup windows.
4. Assign staff.
5. Prepare load checklist.
6. Check unit status: clean, dry, inspected, available.
7. Check stakes/sandbags/tarps/blowers/cords/hoses/balls/games.
8. Print or save dispatch sheet.
9. Run event safety gate.

Multiple bookings same day:

- route by setup deadlines and geography
- load trailer in reverse delivery order
- avoid pickup conflicts
- add buffer time

## 7. Public event workflow

Trigger: school, daycare, church, festival, community association, municipality, large attendance, public park.

Steps:

1. Estimate crowd and age range.
2. Identify organizer and responsible contact.
3. Confirm location permission.
4. Confirm power/water.
5. Confirm insurance certificate need.
6. Determine attendants.
7. Review weather/safety risk.
8. Require owner approval for unusual risk.
9. Draft professional quote/proposal.

## 8. Weather workflow

1. Check forecast and wind.
2. Classify: Green / Yellow / Red.
3. If Green: proceed with normal caution.
4. If Yellow: owner review and customer update.
5. If Red: do not operate; apply policy.
6. Use written communication.
7. Update CRM with decision and reason.

## 9. Post-event workflow

Same day:

- thank customer
- check satisfaction
- note any issue

Next day:

- review request
- photo/testimonial request if appropriate

After review/photo:

- save screenshot
- update CRM
- add to social proof library
- schedule next-year reminder if birthday/school recurring event

## 10. Website update workflow

1. Confirm source-of-truth data.
2. Update copy/pricing/package logic.
3. Check mobile conversion.
4. Check CTA links.
5. Check SEO title/meta/FAQ.
6. Run website quality gate.
7. Push only after owner approval if pricing/policy changed.
