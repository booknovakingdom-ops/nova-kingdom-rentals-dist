# Department Workflow Map

## 1. New Lead to Quote
```text
CEO Orchestrator
→ Sales Booking Manager
→ Lead Intake Agent
→ Availability Manager
→ Package Recommender
→ Quote Builder
→ Finance Revenue Manager if travel/staff/discount/large package
→ Safety Compliance Manager if school/daycare/public/water/weather risk
→ Customer Experience Manager for final wording
→ Quality Control Manager
→ Owner sends or approves
```

## 2. Quote to Deposit
```text
Sales Booking Manager
→ Booking Pipeline Manager
→ Deposit Chaser / Objection Handler
→ Customer Experience Manager
→ Quality Control Manager
→ CRM update
```

## 3. Deposit to Confirmed Booking
```text
Sales Booking Manager
→ Documents Legal Manager for invoice/agreement/waiver
→ Operations Dispatch Manager for calendar/load/staffing
→ Safety Compliance Manager for safety risk
→ Customer Experience Manager for confirmation message
→ Quality Control Manager
```

## 4. Event-Day Operations
```text
CEO Orchestrator
→ Safety Compliance Manager for weather/safety status
→ Operations Dispatch Manager for route/load/staffing
→ Documents Legal Manager confirms waiver/payment/invoice status
→ Quality Control Manager flags missing risk
→ Owner/staff execute
```

## 5. Post-Event Growth Loop
```text
Operations Dispatch Manager confirms pickup/condition
→ Safety Compliance Manager if incident/damage
→ Customer Experience Manager sends thank-you/review request
→ Marketing Growth Manager requests photos/testimonials with permission
→ Finance Revenue Manager logs revenue/profit
→ Booking Pipeline Manager schedules repeat-booking reminder
```

## 6. Website Update
```text
CEO Orchestrator
→ Website SEO Manager
→ Finance Revenue Manager if pricing/package/offer changes
→ Marketing Growth Manager if campaign/brand implications
→ Tech Automation Manager for Codex/GitHub/Hostinger prompt
→ Quality Control Manager
→ Owner approval before live change
```

## 7. Complaint / Refund / Damage
```text
CEO Orchestrator
→ Customer Experience Manager for tone
→ Safety Compliance Manager for facts/risk/incident
→ Documents Legal Manager for agreement/policy language
→ Finance Revenue Manager for refund/credit/cost
→ Quality Control Manager
→ Owner approval required
```
