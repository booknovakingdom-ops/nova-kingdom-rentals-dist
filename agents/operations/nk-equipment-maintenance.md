# Nova Kingdom Equipment Maintenance Tracker

## Post-Event Cleaning Checklist (After Every Booking)
[ ] Unit fully deflated and air expelled | [ ] Interior brushed out (dirt, grass, leaves)
[ ] Exterior wiped down (mud, scuff marks) | [ ] Seams and stitching checked
[ ] Slide surface cleaned and dried | [ ] Blower cleaned, intake clear
[ ] Seams: no tears/separation | [ ] Vinyl: no punctures (check with flashlight)
[ ] Blower: powers on, inflates fully, no unusual noise | [ ] Stakes: all present
[ ] Extension cords: no damage | [ ] Unit folded dry before storage
RESULT: PASS / NEEDS REPAIR (describe issue + action + est. repair date)

## Pre-Event Inspection (Day Before Every Booking)
[ ] Full inflation test (inflate, check pressure at 5 min) | [ ] All seams — no new damage
[ ] Blower operating | [ ] Slide: clean and undamaged | [ ] All accessories packed
RESULT: GO / NO-GO

## Common Repairs
Small puncture → Patch kit (vinyl patch + PVC glue), dry 24 hrs before use
Seam separation → PVC cement, clamp overnight
Blower issue → Check capacitor first (common failure)

## Seasonal Maintenance
Start of season (April): All units inflated and inspected | Blowers tested | Patch kit restocked | Insurance confirmed
End of season (September): Deep clean all units | Fully dried before storage | All units inventoried
