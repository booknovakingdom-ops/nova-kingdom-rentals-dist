## Post-Event Cleaning Checklist (Run After Every Booking)

```
UNIT: _______________  DATE: ____________

DEFLATION & PACKING
[ ] Unit fully deflated and air expelled
[ ] All parts accounted for (slides, obstacles, entrance)
[ ] Blower disconnected and inspected

CLEANING
[ ] Interior brushed out — dirt, grass, leaves removed
[ ] Exterior wiped down — mud, scuff marks cleaned
[ ] Seams and stitching checked for debris
[ ] Slide surface cleaned and dried
[ ] Blower unit cleaned — intake clear of debris

INSPECTION
[ ] Seams: no tears, separation, or fraying visible
[ ] Vinyl panels: no punctures or holes (check with flashlight)
[ ] Blower: powers on, inflates fully, no unusual noise
[ ] Stakes/anchors: all present and undamaged
[ ] Extension cords: no damage, proper gauge

[ ] PASS — Ready for next booking
[ ] NEEDS REPAIR — Note below:
  Issue: _______________________________________
  Action required: _____________________________
  Est. repair date: ____________________________

STORAGE
[ ] Unit folded neatly per storage protocol
[ ] Stored dry — no moisture trapped inside
[ ] Blower stored with unit
[ ] Accessories (stakes, cords) stored with unit
```

**Common repairs:**
- Small puncture/hole → Patch kit (vinyl patch + PVC glue), dry 24 hrs before use
- Seam separation → PVC cement, clamp overnight
- Blower issue → Check capacitor first (common failure), replace if needed
- Stake bent/broken → Replace immediately — safety item


---

## Critical Rules
- **Never deliver a unit that hasn't passed pre-event inspection.** One equipment failure mid-event destroys trust and the review score.
- **Units must be completely dry before storage.** Moisture causes mould and deterioration.
- **Keep a repair log.** Patterns (same seam failing repeatedly) signal a unit needs retirement or professional repair.
- **Blowers are mission critical.** Keep a spare blower if budget allows — a blower failure mid-event is a nightmare.
- **Photo document any damage** at pickup and after return — protects against customer damage disputes.
