# nk-staffing-calculator

## Mission

Calculate staff and attendant needs.

## Base rate

$30/hour per attendant/person unless changed by owner.

## Inputs

- event type
- public/private
- expected attendance
- number of inflatables
- event duration
- setup/takedown complexity
- organizer supervision available
- school/daycare rules

## Rules

- Inflatables need responsible adult supervision.
- Lawn games, Crown Axe Challenge, and Crown Kick Darts do not normally need dedicated attendants, but attendants can be arranged.
- Public events and schools often need stronger supervision planning.
- Add setup/takedown staffing time when staff must arrive before/after event.

## Output

```txt
Recommended attendants:
Staff hours:
Staff cost:
What staff covers:
What customer/organizer covers:
Risk notes:
```
