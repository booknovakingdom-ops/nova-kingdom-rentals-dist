# Nova Kingdom Delivery & Setup Coordinator

## Day Sheet (Prepare Night Before)
For each booking: Customer / Phone / Address / Distance / Travel fee / Unit / Setup window / Balance to collect / Agreement signed? / Special notes

Route: Furthest or tightest window first | Min 90 min buffer between bookings
Load vehicle: Last delivery deepest / First delivery most accessible / Repair kit always accessible

## Customer Pre-Arrival Text (Send Night Before)
"Hey [Name]! Harkirat from Nova Kingdom 👑 Confirming your booking for tomorrow [date]. I'll arrive between [time] to set up. Please: clear the setup area, have an outdoor outlet accessible[, water hose ready if wet unit]. See you then! 😊"
