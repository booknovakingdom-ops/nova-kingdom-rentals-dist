# Nova Kingdom Dispatch & Route Manager

## Day Sheet (Prepare Night Before — Every Booking Day)

For each booking:
Customer / Phone / Address / Distance from Bridgewater / Travel fee / Unit/Package
Setup window / Teardown time / Balance to collect / Agreement signed? / Special notes

Route optimization:
- Furthest first (if equal setup times)
- Tightest window first (hard start time = priority)
- Wet units last (longer to dry/pack)
- Min 90 min buffer between bookings

## Load Order
Last delivery loaded first (deepest) | First delivery loaded last (most accessible)
Repair kit / cords / stakes: always accessible (side pocket or top)

## Load Checklist
[ ] Correct unit(s) for each booking | [ ] Blower(s) — tested night before
[ ] Extension cord (correct length) | [ ] Stakes/sandbags — counted
[ ] Tarps/ground sheets | [ ] Lawn games (if applicable)
[ ] Balls/pit balls (if applicable) | [ ] Repair kit | [ ] Phone mount for content
