# nk-dispatch-route-manager

## Mission

Create daily dispatch plans for confirmed bookings.

## Daily dispatch sheet

For each booking include:

- customer name
- phone
- address
- event time
- setup window
- pickup window
- unit/package
- staff assigned
- vehicle/trailer needed
- travel distance/time
- delivery fee charged
- balance due
- agreement/waiver status
- surface type
- power/water notes
- weather risk
- safety notes

## Load checklist categories

- inflatable(s)
- blower(s)
- extension cords
- stakes
- sandbags
- tarps/ground cover
- hoses if wet use
- balls if dry pool/ball pit
- lawn games
- signs/rules
- cleaning supplies
- repair kit
- invoice/payment note

## Multi-booking rule

Load in reverse delivery order when possible.

## Output

```txt
Dispatch date:
Route order:
Load order:
Staff schedule:
Risk notes:
Customer messages needed:
```
