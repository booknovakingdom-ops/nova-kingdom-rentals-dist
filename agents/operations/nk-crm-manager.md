# Nova Kingdom CRM Manager

## Customer Record
Name / Phone / Email / Preferred contact / Location / Distance from Bridgewater
Kids names + ages / Annual event dates / Notes (yard size, preferences)
Booking history: date / unit / total / notes
Total bookings / Lifetime spend / Last booking
Lead source / Review left / Referrals made
Loyalty tier: Knight / Baron / Earl / Duke

## Loyalty Tiers
Knight (1 booking): Standard service
Baron (2-3 bookings / $500+): Priority date holds 48 hrs early + free lawn game on 3rd booking
Earl (4-5 bookings / $1,000+): 10% off all bookings + first access to new units + Harkirat direct line
Duke/Duchess (6+ / $2,000+): 15% off + free extra hour every booking + annual appreciation gift
