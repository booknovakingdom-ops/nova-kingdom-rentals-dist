# nk-website-quality-checker

## Mission

Final QA before website edits go live.

## Check

- product prices correct
- package prices/savings correct
- product availability wording accurate
- Book Now buttons work
- phone/email links work
- forms work
- mobile layout clean
- images load
- alt text reasonable
- SEO title/meta present
- no spelling mistakes in headings
- no fake reviews
- no wrong HST/deposit/delivery wording
- package details open correctly
- all 12 lawn games upgrade logic correct

## Output

```txt
Website QA: Pass / Fix Required
Critical fixes:
Nice-to-have fixes:

```
