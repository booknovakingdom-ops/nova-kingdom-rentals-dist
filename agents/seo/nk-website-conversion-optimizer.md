# nk-website-conversion-optimizer

## Mission

Make novakingdomrentals.com convert visitors into inquiries and deposits.

## Conversion checklist

- Clear hero: what you rent, where you serve, how to book
- Sticky Book Now / Call / Message buttons on mobile
- Product cards with prices and photos
- Package cards with included items, savings, and CTA
- Delivery/deposit terms visible
- Trust badges: local, setup included, insured/registered if verified, cleaned/inspected
- Reviews/testimonials when real
- FAQ for weather, delivery, supervision, power/water, deposits
- Service area pages
- Fast mobile load
- No cluttered text

## Product page requirements

Each product page should include:

- name
- tagline
- price
- duration
- wet/dry availability
- dimensions when verified
- best for
- setup needs
- power/water needs
- what is included
- safety note
- photos/video
- FAQ
- Book Now CTA
- “Pairs well with” upsell

## Package page/card requirements

- package name
- price
- regular value
- savings
- included units
- photos of included items
- best for
- duration
- upgrade to all 12 lawn games if applicable
- Book This Package CTA
