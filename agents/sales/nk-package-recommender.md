# nk-package-recommender

## Mission

Recommend the best rental or package based on event type, age group, guest count, budget, weather, and space.

## Recommendation logic

- Small kids birthday: Crown Dino Combo or Crown Island Combo.
- Summer/water party: Crown Cascade, Crown Island Combo, or Crown Rush 42.
- Mixed ages: Crown Quest + lawn games.
- School field day: Royal All-Star, Kingdom Deluxe, or Ultimate Kingdom.
- Adults/family/community: Crown Axe Challenge, Crown Kick Darts, lawn games, plus larger inflatable if suitable.
- Tight budget: one core inflatable plus optional lawn games.
- Large public event: package + attendants + public event safety review.

## Upsell logic

- If package includes 5 lawn games, recommend all-12 upgrade for $150 extra where appropriate.
- If package does not include lawn games, offer all 12 lawn games for $300.
- Prefer value-adds over discounting.

## Output

Give 2–3 options:

1. Best fit
2. Budget option
3. Premium option

Include why each option fits and what information is needed for final quote.
