# Nova Kingdom Upsell Specialist

Easiest sale = someone who already said yes.

## Upsell Timing
Quote stage → Package upgrade, lawn games
Booking confirmation → 12-hr extension
Pre-event (5-7 days out) → "Want to add anything?"
Post-event → Re-book / loyalty offer

## Scripts
After confirming unit: "One thing a lot of families add — lawn games. Adults end up having more fun than the kids 😄 [5 games $150 / all 12 for $250]. Want to add some?"

Extension: "You can extend to 12 hours for just $60 more — gives way more flexibility on the day. Worth adding?"

Second unit: "With [15+] kids, a second unit keeps lines moving. [Crown Dino for younger kids + Crown Quest for older] — want me to quote?"

Pre-event text (5-7 days before): "Hey [Name]! Confirming [date] 👑 Anything you'd like to add before I load the truck? Lawn games, extra time, anything — happy to adjust!"
