# Nova Kingdom Upsell Specialist

## Business Context
**Nova Kingdom Rentals** — Bridgewater, Nova Scotia
**Pricing reference**: Individual units $210-$450 (8 hr) · Event packages $310-$1,499 (6 hr) · Extension +$60 · Travel: 15 km free, $0.72/km after

The average booking value increases significantly with one upsell. A customer booking a $210 Crown Dino Combo who adds lawn games and a 4-hour extension becomes a $330+ booking — 57% more revenue, same delivery trip.

## Upsell Scripts

### 1. Package Upgrade (at Quote Stage)
When a customer asks for a single unit, check if a package saves them money OR adds more value:

```
Before I send your quote, quick question — are you planning any activities for adults too, or just the bouncy castle for the kids?

[If they say adults too / bigger event]:
In that case, the [Quest & Games / Dino Party Plus] package might actually be better value — it includes the bouncy castle plus lawn games for $[price]. Based on what you've described, it's probably a better fit.

Want me to quote both options so you can compare?
```

### 2. Lawn Game Add-On (Most Common Upsell)
After they confirm a unit:
```
One thing a lot of families add is a lawn game set — Giant Jenga, Cornhole, or Giant Connect 4. The adults end up having more fun than the kids honestly 😄

I can add [specific game] to your rental for $[price]. Want to throw one in?
```

### 3. Extension to 12 Hours (+$60)
After confirming an 8-hour rental:
```
One thing worth knowing — you can extend to 12 hours for just $60 more.

A lot of families find 8 hours is tight once setup time is factored in. The 12-hour option gives you way more flexibility on the day.

Worth adding?
```

### 4. Second Unit (For Larger Events)
When the event has mixed ages or large guest count:
```
How many kids are you expecting? 

[If 15+]: With that many kids, a second unit actually keeps the lines moving — less waiting, more playing. Some families book [Crown Dino Combo for younger kids] + [Crown Quest for older kids] together.

I can give you a multi-unit price if you want to explore that.
```

### 5. Pre-Event "Add Anything?" Text
Send 5-7 days before every booking:
```
Hey [Name]! Just confirming everything is set for [date] 🎉

Quick check — is there anything you'd like to add before the event? Lawn games, an extra hour, or anything else?

Happy to adjust before I load up the truck 👍
```

### 6. Post-Event Loyalty Upsell
After a great event, plant next year:
```
Hey [Name]! Glad to hear the party was a hit 🎉

Just so you know — returning customers get first pick on dates for next year. If you want to lock in the same weekend in 2027, just say the word.

Also — Crown Rush 42 will be available this June if you're planning anything bigger 👀
```

## Critical Rules
- **Offer upsells once, naturally — never pressure.** A pushy upsell destroys trust.
- **Only suggest upsells that genuinely add value** for their specific event.
- **Frame upsells as making their event better**, not as making more money.
- **The extension upsell is the easiest** — almost always worth mentioning.
- **Lawn games upsell works best when you mention adults**, not just kids.
