# Nova Kingdom Visual Asset Manager

## Business Context
**Nova Kingdom Rentals** — Bridgewater, Nova Scotia
**Editing tools**: CapCut (video), phone camera (photos)
**AI video**: Kling AI (for product visuals when no real footage exists)
**Storage**: Google Drive folder — "Nova Kingdom Content Library"

---

## Event Day Photography Protocol

Every booking is a content opportunity. Here's what to capture at every setup:

### At Delivery/Setup (Always — No Permission Needed for Equipment)
```
[ ] Wide shot: empty yard → inflatable fully set up (before/after or time-lapse)
[ ] Close-up: unit entrance, slide, key features
[ ] Scale shot: stand next to the unit to show size
[ ] Detail shot: stitching, quality, clean presentation
[ ] Setup in progress: action shot of inflation
```

### During the Event (Requires Customer Permission)
Ask at setup: "Would you mind if I grab a few quick photos/videos for our social media? I'll tag you if you'd like and always ask before posting."

If YES:
```
[ ] Kids' faces seeing the unit for the first time (reaction shot)
[ ] Kids playing/jumping — wide shot
[ ] Close-up of joyful face (with parent permission)
[ ] Adults playing lawn games if applicable
[ ] Group shot of the setup in full swing
[ ] 15-30 second video clip of activity
```

### At Teardown (Always)
```
[ ] Photo of the yard after pickup — clean, nothing left behind
[ ] Unit folded and loaded — shows professionalism
```

## Photo Quality Standards

Every photo used publicly must pass this check:
```
[ ] In focus — no blur
[ ] Well lit — natural daylight preferred, no dark shadows
[ ] Horizontal (landscape) for website/ads, vertical (portrait) for Instagram/Stories
[ ] No strangers in background
[ ] Equipment clean and undamaged
[ ] No personal items (cars, house numbers, private info) unless deliberately chosen
[ ] Customer children: explicit permission before posting
```

**Reject and retake** anything that doesn't pass. One great photo beats ten mediocre ones.

## Kling AI — When to Use It

Use Kling AI only when:
- A new unit (Crown Rush 42) has no real footage yet
- Weather prevented capturing good content at a booking
- You need a specific visual concept that's hard to capture in real life

Always prefer real footage — it builds more trust and generates better engagement than AI video.

---

## UGC (User-Generated Content) Collection

When customers send you photos or post tagged content:

1. Save to Google Drive immediately: `Events/2026-Bookings/[date]-[name-UGC]/`
2. Send to Review Specialist: screenshot anything with a positive caption
3. Send to Social Media Manager: queue for repost within 48 hours
4. Ask permission before reposting if not already tagged publicly

**UGC Request to customers** (at setup):
```
"If you get any great shots today, feel free to tag us @novakingdomrentals — we love seeing the parties! And if you send them our way, we're happy to repost and give you a shoutout 😊"
```

---

## Critical Rules
- **Never post a blurry or dark photo publicly.** It undermines the premium brand.
- **Never post children's faces without explicit permission** from the parent.
- **Always watermark or add handle to key assets** before posting — protects content from being reposted without credit.
- **Empty storage is a content emergency.** If you have less than 2 weeks of queued content, shoot more.
- **Real event content always outperforms AI content.** Capture at every booking.
