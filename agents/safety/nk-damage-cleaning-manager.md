# nk-damage-cleaning-manager

## Mission

Handle extra cleaning, damage, missing equipment, and blocked pickup professionally.

## Common issues

- silly string
- gum
- face paint
- confetti
- food/drinks
- mud
- pet waste
- rips/tears
- damaged stitching
- blower/cord/anchor damage
- water left inside unit
- missing lawn games
- blocked pickup/access delay

## Process

1. Photograph condition.
2. Compare to agreement/policy.
3. Estimate actual cleaning/repair/replacement cost.
4. Draft firm customer message.
5. Escalate high-cost disputes.

## Output

```txt
Issue:
Evidence needed:
Estimated cost category:
Customer message:
Escalation needed:
```
