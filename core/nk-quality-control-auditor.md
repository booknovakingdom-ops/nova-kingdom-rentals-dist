# Nova Kingdom Quality Control Auditor

Run before sending: any quote, booking confirmation, invoice, social post, ad, website update, contract

## Quote QC Check
[ ] Customer name (not "Dear Customer") | [ ] Date and event details correct
[ ] Unit/package name matches source-of-truth exactly | [ ] Price matches source-of-truth
[ ] Travel fee: (distance - 15) × $0.72 | [ ] Extension +$60 if applicable
[ ] Credit card +5% if applicable | [ ] Lawn games: $150 (5 games) / $250 (all 12) / +$100 upgrade
[ ] HST NOT added | [ ] Deposit = total × 0.30 | [ ] Balance = total - deposit
[ ] Payment: e-transfer booknovakingdom@gmail.com | [ ] "Setup and takedown included"
[ ] Crown Rush 42: NOT before June 2026 | [ ] Clear next step | [ ] Warm human tone

## Booking Confirmation QC
[ ] Name / Date (day+date+year) / Unit / Address correct | [ ] Setup time correct
[ ] Total / deposit / balance correct | [ ] Balance due timing (day-of at delivery)
[ ] Contact: 902-990-0005 | [ ] Agreement/waiver mentioned | [ ] No unavailable promised

## Social/Ad QC
[ ] Crown [Name] format | [ ] Correct price | [ ] Location tag | [ ] Contact info
[ ] No "guaranteed availability" or "lowest price" | [ ] No blurry/dark images
[ ] No children without permission | [ ] No competitors | [ ] Max 4 emojis
[ ] Spelling/grammar | [ ] CTA present

## Auto-Fail (Do Not Send)
Wrong price | HST charged when not registered | Crown Rush 42 before June 2026
Deposit under 30% | Admission of fault/liability | "AI" mention to customer
Missing customer name in personal message | Availability confirmed without calendar check

## Common Catches
Wrong travel fee → recalculate: (km-15) × $0.72
Missing 5% card fee → always ask payment method first
Deposit math → total × 0.30
Tone too formal → read it aloud, does it sound like a person?
Missing next step → every message ends with question or action
